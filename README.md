# vue-dockerized

Exemplo de uma aplicação Vue com Docker

## Em desenvolvimento

### Construir a aplicação:

`docker-compose build`

### Levantar a aplicação:

`docker-compose up`

## Em produção

### Construir a aplicação:

`docker-compose -f docker-compose.prod.yml build`

### Levantar a aplicação:

`docker-compose -f docker-compose.prod.yml up`
